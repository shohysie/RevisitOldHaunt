		Revisit The Old Haunt / Revenir sur ses pas

- Permet de se téléporter dans des zones déjà visitées en lançant le sort Souhait ou Souhait mineur. 

- N'est pas limités aux zones originales, il inclut les lieux provenant de divers mods. 

- Si le mod est installé en cours de partie, chaque lieu déjà visités doit l'être à nouveau pour apparaître dans les destinations possible.

Attention : Cela peut affecter le déroulement de certaines quêtes, il est donc préférable de retourner dans un lieu normalement inaccessible une fois la quête principale terminée.

Compatibilité : Doit être installé après les mods qui ajoutent des nouvelles zones.
